### Install the CLI

The DesterLib CLI makes it easy to set up and manage your media server.

**macOS/Linux:**

```bash

```

**Windows (PowerShell):**

```powershell
iwr -useb https://raw.githubusercontent.com/DesterLib/desterlib/main/packages/cli/install.ps1 | iex
```
