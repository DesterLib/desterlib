# Dester Web App

Frontend web application for the Dester media library built with React, TypeScript, and Vite.

## Development

```bash
# Install dependencies (from monorepo root)
pnpm install

# Start dev server
pnpm --filter web dev
```

## Build

```bash
pnpm --filter web build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- @dester/api-client for API communication
