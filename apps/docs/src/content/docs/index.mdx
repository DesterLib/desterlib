## Project Components

### Backend API
The engine that does all the work:
- Organizes your media files
- Streams videos to your devices
- Remembers what you're watching
- Manages the database

### Mobile & Desktop App
The way you watch on your devices:
- Browse your media collection
- Play videos smoothly
- Manage settings
- Cross-platform support
